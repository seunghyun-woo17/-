# Avikus 구매관리 시스템 MVP

Avikus 사내 바코드 기반 자재 입고 및 재고 관리 프로토타입입니다.

---

## 이 프로그램이 뭔지

- PO(구매발주서) 생성 및 QR 코드 발행
- QR 스캔을 통한 자재 입고 처리
- 입고 검수 리포트 자동 생성
- 재고 DB 조회 (5개 테이블)

현재는 1인 검증용 프로토타입이며, 추후 팀 단위 사용을 위한 서버 이전 예정입니다.

---

## 어떻게 실행하는지

브라우저에서 아래 주소로 바로 접속 가능합니다:

https://seunghyun-woo17.github.io/-/avikus_mvp_prototype.html

또는 `avikus_mvp_prototype.html` 파일을 다운로드 후 브라우저에서 직접 열어도 됩니다.

| 파일 | 설명 |
|------|------|
